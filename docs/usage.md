# Usage Guide

## Getting Started
1. **Login**: Navigate to `http://localhost:5250` and enter your `APP_PASSWORD`.

## Workflow

### 1. Build Components
Go to **Components** to create reusable assets:
- **Characters**: Define base appearance (hair, eyes, body).
- **Wardrobe**: Create outfits or individual pieces.
- **Poses/Scenes**: Define composition and environment.

### 2. Compose Prompt
Navigate to **Builder**:
1. Select a **Character**.
2. Mix & match **Wardrobe**, **Jewelry**, and **Props**.
3. Choose a **Pose** and **Scene**.
4. Review the generated JSON structure in the preview pane.

### 3. Generate
- Click **Generate** to send to queue.
- Monitor progress in the status bar.

### 4. Analyze (Optional)
- Use **Analyze** tab to upload an existing image.
- AI extracts prompt components to save for future use.
