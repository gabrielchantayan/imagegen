# Features

## Core
- **Structured Prompt Builder**: Compose JSON prompts using reusable components (Characters, Wardrobes, Poses, Scenes).
- **Gemini Integration**: Native support for Google's `gemini-3-pro-image-preview` model.
- **Queue System**: Manages concurrent generation requests.

## Tools
- **Image Analysis**: Upload an image to reverse-engineer its prompt structure and create new components.
- **History Gallery**: View past generations with full prompt metadata and regeneration options.
- **Component Manager**: CRUD interface for all prompt building blocks (Jewelry, Backgrounds, Camera settings).

## Admin
- **Dashboard**: Track usage statistics, generation counts, and model performance.
- **User Management**: Simple password-based access control.
